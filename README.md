## 🛡️ Overview

**CUSTOS** is a cinematic, fully-responsive landing page for an autonomous AI security startup. Built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no dependencies — the site tells a powerful story of intelligent, offline-first security that detects, understands, and responds to threats automatically.

The website was built around a true story: a family who had two working CCTV cameras the night they were robbed — and yet no one knew until the next morning.

---

## ✨ Features

### 🎨 Design System
- **Pure black** (`#000000`) cinematic base with electric-cyan (`#00C8FF`) and deep-blue (`#0057FF`) accents
- **Glass morphism cards** with backdrop blur, inner glow, and hover lift effects
- **Ambient glow blobs** for atmospheric depth across all sections
- **Fonts:** `Syne` (headings) + `DM Sans` (body) — both via Google Fonts CDN
- Fully **mobile-first responsive** — breakpoints at 768px (tablet) and 480px (mobile)

### 🎬 Animations
| Animation | Description |
|-----------|-------------|
| Scroll Reveal | Every section element fades + slides up on viewport entry via Intersection Observer |
| Pulsing Dot | Hero badge cyan dot pulses scale + opacity on 1.5s loop |
| Traveling Dot | Animated cyan dot moves along connecting lines in architecture & escalation diagrams |
| Camera Sweep | Hero floor-plan diagram shows animated camera cone sweeps |
| Detection Flicker | AI bounding boxes appear/disappear to simulate live AI inference |
| Grid Parallax | Hero SVG grid shifts subtly on scroll via `requestAnimationFrame` |
| Escalation Stagger | Decision tree nodes reveal one-by-one with 0.3s delays on scroll |
| Globe Nodes | Map dots spawn and pulse outward on a 2s loop |
| Tab Crossfade | AI Intelligence tab switcher with 0.3s opacity transitions |

## 📦 Deployment

### ▲ Deploy to Vercel (Recommended)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Click **Deploy** — done ✅

Your site will be live at `https://custos.vercel.app` (or your custom domain).

Every `git push` to `main` triggers an **automatic redeploy**.

---

### 🐙 Deploy to GitHub Pages

1. Go to your repo → **Settings → Pages**
2. Under **Branch**, select `main` → `/ (root)` → **Save**
3. Your site will be live at:
   ```
   https://YOUR_USERNAME.github.io/custos-website/
   ```
